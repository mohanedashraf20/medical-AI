# AI Bylaws Chatbot

## Features
- Smart Search (RAG)
- ChatGPT-like UI
- Arabic & English support
- Article detection
- Source display
- Memory

## Run Backend
cd backend
pip install -r requirements.txt
python app.py

## Run Mobile App
cd mobile-app
npm install
npm start

## Notes
- Replace YOUR_API_KEY
- Replace YOUR_IP with your local IP
